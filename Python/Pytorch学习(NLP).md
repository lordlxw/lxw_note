pytorch是基于Numpy的科学计算包

两大优势：

1、能用到GPU的强大算力

2、向用户提供最大灵活性和速度



pip和conda

> **选择 pip**：当只需管理 Python 包且不涉及多个环境管理时，pip 是一个简单和常用的选择。
>
> **选择 conda**：当需要管理不仅限于 Python 包的软件包、管理多个环境以及在不同平台上保持一致性时，conda 提供了更强大和灵活的功能。
