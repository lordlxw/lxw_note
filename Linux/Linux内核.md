# Linux内核

## 一、体系结构

> 操作系统就是 帮助用户从用户态-》内核态（做事）-》返回用户态
>
> 用户态切换到内核态——用到的是kernel文件夹里的asm.s、sys_call.s 两个链接文件做好切换的准备工作，
>
> sys.c——具体实现某些函数
